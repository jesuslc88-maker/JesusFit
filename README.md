# JesusFit Tracker 💪

App de seguimiento nutricional personalizada.

## Cómo instalar en GitHub Pages

1. Crea cuenta en https://github.com si no tienes
2. Crea repositorio nuevo llamado `jesusfit`
3. Sube estos archivos: index.html, manifest.json, sw.js
4. Ve a Settings → Pages → Source: main branch
5. Tu app estará en: https://TU_USUARIO.github.io/jesusfit
6. Abre esa URL en Chrome Android → menú 3 puntos → Añadir a pantalla de inicio
